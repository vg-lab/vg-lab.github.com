11 ## Introduction
12 
13 FiReS is a library that allow registering type-independent properties in
14 objects. FiReS also provides tasks that can use the registered properties for
15 filtering, search, sorting operations over a set of objects.
16 
17 ## Dependencies
18 
19 Boost.Any, Boost.Lexical_Cast, Boost.property_tree and Boost.Spirit are the
20 only required dependencies.
21 
22 OpenMP and Boost's Unit Test Framework are optional.
23 
24 ## Building
25 
26 FiReS has been succesfully built and used on Ubuntu 16.04, Mac OSX
27 Yosemite and Windows 7/8/10 (Visual Studio 2015 Win64). The following steps
28 should be enough to build it:
29 
30 ```bash
31 git clone https://github.com/vg-lab/FiReS.git
32 mkdir FiReS/build && cd FiReS/build
33 cmake .. -DCMAKE_BUILD_TYPE=Release
34 make
35 ```

37 ## How to use it
38 
39 ### Registering properties
40 ```c++
41 class MyClass : public fires::Object
42 {
43 };
44 ...
45 MyClass myObject;
46 myObject.registerProperty( "property1", 3.0f );
47 myObject.registerProperty( "property2", int( 2 ));
48 ```
49 
50 ### Retrieving properties
51 ```c++
52 float v1 = myObject.getProperty( "property1" ).value( );
53 int v2 = myObject.getProperty( "property2" ).value( );
54 ```
55 
56 ### Creating notifier/observer
57 ```c++
58 class MyClass : public fires::Object
59 {
60 public:
61  FIRES_OBSERVER_CALLBACK( observerCallback, notifier, propertyLabel )
62  {
63  std::cout << label( ) << ": Hey!" << " object " << notifier->label( )
64  << " updated " << propertyLabel << " to "
65  << notifier->getProperty( "p1" ).value< int >( ) << std::endl;
66  }
67 };
68 ...
69 MyClass o1, o2;
70 o1.label( ) = "o1";
71 o2.label( ) = "o2";
72 o1.registerProperty( "p1", 1 );
73 fires::ObserverManager::addObserver( &o1, "p1", &o2, &MyClass::observerCallback );
74 o1.setProperty( "p1", 2 );
75 // output:
76 // o2: Hey! object o1 updated p1
77 ```
78 
79 ### Sorting
80 ```c++
81 class MyClass : public fires::Object
82 {
83 };
84 
85 int main ( )
86 {
87 
88  MyClass o1, o2, o3, o4;
89 
90  fires::Sort sorter;
91  fires::SortConfig sortConfig;
92  fires::ScalarPropertySorter< float > sfsf;
93  fires::Objects objs;
94 
95  // Register property
96  o1.registerProperty( "p1", 3.1f );
97  o2.registerProperty( "p1", 4.1f );
98  o3.registerProperty( "p1", 1.2f );
99  o4.registerProperty( "p1", 3.1f );
100 
101  // Create the objects container
102  objs.addList( { &o1, &o2, &o3, &o4 } );
103 
104  // Configure sort config
105  sortConfig.addProperty( "p1", &sfsf );
106 
107  // Eval the task
108  sorter.eval( objs, sortConfig );
109 }
110 ```
