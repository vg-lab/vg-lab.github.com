4 ## Introduction
5 NeuroLOTs is a set of libraries and tools for generating neuronal meshes and for
6 visualizing them at different levels of detail using GPU-based tessellation.
7 Providing tools for the generation of 3D polygonal meshes that approximate the
8 membrane of neuronal cells, from the morphological tracings that describe the
9 morphology of the neurons. The 3D models can be tessellated at different levels
10 of detail, providing either homogeneous or adaptive resolution along the model.
11 The soma shape is recovered from the incomplete information of the tracings,
12 applying a physical deformation model that can be interactively adjusted.
13 
14 See [NeuroTessMesh web page](http://vg-lab.es/neurotessmesh/) and
15 [NeuroTessMesh manual](http://vg-lab.es/neurotessmesh/NeuroTessMeshUserManual.pdf)
16 for a complete description and sample data tests.

30 ## Building
31 
32 neurolots has been successfully built and used on Ubuntu 16.04 LTS, Mac OSX High
33 Sierra and Windows 10 64-bits with Visual Studio 2013. Please ensure that you
34 build the Release version in order to get the best performance possible.
35 
36 ```bash
37 git clone https://github.com/vg-lab/neurolots
38 mkdir neurolots/build && cd neurolots/build
39 git submodule update --init
40 cmake .. -DCLONE_SUBPROJECTS=ON
41 make
42 ```
